# Natural-language-processing-project

Legal NER for U.S. Supreme Court Opinions

- Allen Guo

## Project Description
This project builds a multi-stage named entity recognition pipeline for U.S. Supreme Court opinions.
The system targets legal-domain entity types such as PARTY, JUDGE, COURT, STATUTE, CITATION, ORG, and DATE.
